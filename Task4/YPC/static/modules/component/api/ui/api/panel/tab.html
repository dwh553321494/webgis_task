# 属性

# tabs

  • 类型: Array
  • 非侦听属性
  • 描述: 显示切换的页卡
[
    {
      "type": "highlight",
      "title": "高亮",
      "icon": "mapgis-target-lock",
    },
    {
      "type": "oid",
      "title": "OID查询",
      "icon": "mapgis-bullseye",
    },
    {
      "type": "properties",
      "title": "属性查询",
      "icon": "mapgis-table",
    }
]

  • 载荷 payload
