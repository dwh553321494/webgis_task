# 综合查询

# Props

# logo

  • 类型: String
  • 非侦听属性
  • 描述: 查询框的左侧图标

# districtName

  • 类型: String
  • 非侦听属性
  • 描述: 查询框的左侧文字

# defaultMarkerIcon

  • 类型: String
  • 非侦听属性
  • 描述: 标注默认图标

# selectedMarkerIcon

  • 类型: String
  • 非侦听属性
  • 描述: 标注选中图标

# geoJSONExtent

  • 类型: Object|GeoJson
  • 非侦听属性
  • 描述: 查询范围

# 事件

# onClose

  • 描述: 点击关闭按钮的回调函数

# onSearch

  • 描述: 点击查询按钮的回调函数

# current-result

  • 描述: 当前面板查询结果的回调函数
  • 回调参数: object
  • 示例:
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "NAME": "立君酒楼",
        "ADDRESS": "湖南省常德市鼎城区唐家铺乡金泉山村",
        "TELEPHONE": "",
        "PAC": "430703",
        "markerId": "place-name-0"
      },
      "geometry": {
        "type": "Point",
        "coordinates": [111.73674642054412, 28.78661878618419]
      },
      "bound": {
        "xmin": 111.73674642054412,
        "ymin": 28.78661878618419,
        "xmax": 111.73674642054412,
        "ymax": 28.78661878618419
      }
    }
  ]
}

# select-markers

  • 描述: 当前选中的条目 id 数组
  • 回调参数: Array
  • 示例:
["place-name-0"]

# select-markers

  • 描述: 当前选中的查询结果的 id 数组
  • 回调参数: Array
  • 示例:
["place-name-0"]

# change-cluster

  • 描述: 点击开启聚合模式的开关回调函数
  • 回调参数: Boolean
  • 参数说明: 是否开启聚合模式 true|false

# open-attribute-table

  • 描述: 展示模式为属性表展示时，查询属性表的回调函数
  • 回调参数: Object

# remove-attribute-table

  • 描述: 关闭属性表回调函数
  • 回调参数: String
  • 参数描述: 当前关闭属性表面板的 id

# color-cluster

  • 描述: 聚合模式下，当前聚合面板，标注点的颜色

  • 描述: 卡片默认内容槽，点击左侧 logo 弹出面板
