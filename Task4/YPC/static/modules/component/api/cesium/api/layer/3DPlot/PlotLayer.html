# PlotLayer

mapgis-3d-plot-layer
行业标绘图层组件，标绘图元的容器
点此跳转到示例

# 属性

名称 类型 默认值 描述 是否监听
dataSource String or Object [] 行业标绘图层数据源，url或数据对象 是
vueIndex String or Number 随机数 plotLayer 的唯一标识，随机生成的数字或字符串，不传则自动生成 否
vueKey String 'default' cesium 球体的唯一标识，默认值 default，当分屏时使用此对象标识多个球体 否

# 事件

事件名称 参数 默认值 描述
loaded data {} 图层加载完毕事件

# data

名称 类型 默认值 描述
vueIndex String or Number 随机数 plotLayer 的唯一标识，随机生成的数字或字符串，不传则自动生成
vueKey String 'default' cesium 球体的唯一标识，默认值 default，当分屏时使用此对象标识多个球体

# 示例


<template>
  <mapgis-web-scene style="height:95vh">
    <mapgis-3d-plot-layer @loaded="handleLoaded"
                          :dataSource="dataSource"
                          :vueIndex="vueIndex"
                          :vueKey="vueKey"
    ></mapgis-3d-plot-layer>
  </mapgis-web-scene>
</template>
<script>
export default {
  data() {
    return {
      //行业标绘图层数据源
      dataSource: "",
      //vueIndex，用来标识图层
      vueIndex: undefined,
      //vueKey，用来标识图层
      vueKey: undefined
    };
  },
  methods: {
    //图层加载完毕事件
    handleLoaded(e) {
      //返回vueIndex以及vueKey
      this.vueIndex = e.vueIndex;
      this.vueKey = e.vueKey;
    }
  }
};
</script>
